# 对比分析

## 目标

对比不同类别、时间段或分组之间的数据差异，揭示各维度的表现差异，帮助用户做出数据驱动的决策。

## 前置条件

- 数据必须包含至少 1 个可用于分组的列（类别型列或时间型列）
- 数据必须包含至少 1 个数值型列用于对比
- 分组列的唯一值建议在 2-10 个之间（过多或过少都不适合对比分析）

## 执行步骤

### 步骤 1：数据概览

调用 `analyzeCsv` 工具，参数：
- `csvData`: 原始 CSV 数据
- `analysisGoal`: "对比分析：识别分组维度和数值指标"

获取结果后关注：
- 类别列（inferredType = string）：识别可作为分组维度的列
- 时间列（inferredType = date）：识别可按时间段对比的列
- 数值列（inferredType = numeric）：识别用于对比的指标列
- 各列的 uniqueValues，确定分组数量是否合理

### 步骤 2：统计计算

调用 `calculateStatistics` 工具，参数：
- `csvData`: 原始 CSV 数据
- `metrics`: "mean,median,stddev"

分析结果：
- 比较各数值列的均值差异
- 通过标准差判断各组内的稳定性
- 均值与中位数的差异判断各组是否存在偏移

### 步骤 3（可选）：知识检索

如果涉及行业对比（如不同区域、不同产品线），调用 `searchKnowledge` 工具：
- `goal`: "{行业}{对比维度}分析方法和基准数据"

### 步骤 4：可视化

调用 `generateChart` 工具，参数：
- `chartType`: 推荐图表类型：
  - 2-5 组对比 → `bar`（柱状图）
  - 时间维度对比 → `line`（折线图）
  - 构成占比对比 → `pie`（饼图）
  - 多维对比 → `radar`（雷达图）
- `analysisResult`: "对比分析：{分组维度}×{指标}"
- `csvData`: 原始数据

## 输出格式

按以下结构输出：

```
【对比分析报告】

一、对比维度
分组列：{列名}（{唯一值数}个分组）
对比指标：{数值列名列表}

二、分组统计
| 分组 | {指标1}均值 | {指标1}中位数 | {指标2}均值 | ... |
|------|-------------|---------------|-------------|-----|

三、关键差异
1. 最大差异：{分组A} vs {分组B}，{指标}相差 {倍数/百分比}
2. 异常分组：{分组名}，原因：{描述偏离情况}
3. 趋势差异：{描述各组在趋势上的不同}

四、分析建议
1. {基于对比结果的建议}
2. {需要进一步分析的维度}
3. {可能的优化方向}
```

## 约束

- 不对分组做主观优劣判断，用数据说话
- 如果分组列唯一值为 1（无分组依据），告知用户当前数据不适合做对比分析
- 如果分组过多（> 10），建议用户聚焦 top 5 进行对比
- 对比结论需考虑各组样本量差异的影响
