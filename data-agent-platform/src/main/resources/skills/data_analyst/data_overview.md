# 数据概览生成

## 目标

对数据集进行全面概览，生成包含数据结构、分布特征、缺失情况、基本统计量的综合报告，帮助用户快速了解数据全貌。

## 前置条件

- 数据为 CSV 格式，至少包含表头和 1 行数据
- 接受任意类型的数据（无特定列要求）

## 执行步骤

### 步骤 1：数据解析

调用 `analyzeCsv` 工具，参数：
- `csvData`: 用户提供的原始 CSV 数据
- `analysisGoal`: "数据概览：全面了解数据结构和分布"

获取结果后关注：
- 总行数（totalRows）和总列数（totalColumns）
- 是否被截断（truncated = true 表示超过 1000 行）
- 每列的 inferredType（numeric / string / date / empty）
- 每列的 missingCount 和 missingRate
- 数值列的 min / max / mean
- 每列的 uniqueValues 和 sampleValues

### 步骤 2（可选）：行业知识检索

如果从列名或样本值中识别出特定行业特征，调用 `searchKnowledge` 工具：
- `goal`: "{识别出的行业}数据分析关键指标和方法"

将行业知识作为补充参考融入报告中。

### 步骤 3：分布可视化

调用 `generateChart` 工具，参数：
- `chartType`: 根据数据特征选择：
  - 数值列分布 → `bar`（柱状图展示各列基本统计量）
  - 类别列分布 → `pie`（饼图展示各类别占比）
- `analysisResult`: "数据分布概览"
- `csvData`: 原始数据

## 输出格式

按以下结构输出：

```
【数据概览报告】

一、基本信息
- 数据规模：{行数} 行 × {列数} 列
- 数据截断：{是/否}（超过1000行时截断）

二、列结构分析
{按类型分类列出所有列}

数值列（{n}列）：
| 列名 | 缺失率 | 均值 | 最小值 | 最大值 | 唯一值数 |
|------|--------|------|--------|--------|----------|

文本列（{n}列）：
| 列名 | 缺失率 | 唯一值数 | 样本值 |
|------|--------|----------|--------|

日期列（{n}列）：
| 列名 | 缺失率 | 样本值 |
|------|--------|--------|

三、数据质量评估
- 整体缺失率：{百分比}
- 高缺失列（>20%）：{列名列表}
- 高基数列（唯一值/总行数 > 80%）：{列名列表}

四、关键发现
1. {第一条发现}
2. {第二条发现}
3. {第三条发现}
```

## 约束

- 不对数据做假设，如实反映数据状态
- 数据完全为空时，返回"数据为空，无法生成概览"
- 截断数据需在报告首行标注警告
- 不建议在此阶段做深入统计分析，仅做概览
